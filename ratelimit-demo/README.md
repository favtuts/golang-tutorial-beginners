# Create web application

Create a demo directory containing two files, `limit.go` and `main.go`, and initialize a new Go module
```bash
$ mkdir ratelimit-demo
$ cd ratelimit-demo
$ touch limit.go main.go
$ go mod init example.com/ratelimit-demo
```
